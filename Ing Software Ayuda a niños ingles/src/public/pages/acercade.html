Acerca De Nosotros

¿Quienes Somos?

Somos una página web que apoya en el aprendizaje y/o refuerzo del idioma inglés



Este proyecto está proyectado para que sea utilizado por niños de educación primaria (6-12 años).



Este refuerzo se piensa llevar a cabo mediante unidades de aprendizaje, actividades, recursos externos, juegos, entre otros medios lúdicos.



El proyecto fue desarrollado para la materia Ingeniería de Software en la Escuela Superior de Cómputo (ESCOM) del Instituto Politécnico Nacional.

Herramientas de Desarrollo

El proyecto fue desarrollado por el equipo 4 del grupo 6CV1 en el periodo 2023-1


Está desarrollado en HTML, CSS, JS, PHP y MySQL.
